# 1504 특정한 최단 경로
v1, v2 를 무조건 지나는 최단 경로 구하기
다익스트라 알고리즘 + 조건 추가
조건 처리 로직:
    case 2개로 나눠서 둘 중 최소 선택해서 출력
    case1 -> 1 - v1 - v2 - V 순으로 가는 경우    
    case2 -> 1 - v2 - v1 - V 순으로 가는 경우
    => a번째 노드부터 b번째 노드까지가는 최단 거리 구하는 걸 저 원소들로 한 경우당 세번씩 수행해서 합해주기
       (즉 다익스트라 함수 총 6번 호출)
       그럼 합한 값 두 개 나오겠지. 그거 둘 중 최소값 출력
